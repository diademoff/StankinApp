# StankinApp

Приложение для просмотра расписания МГТУ "СТАНКИН"

## Реализованный функционал

### База данных
- Реализована база данных SQLite с использованием Room
- Созданы таблицы для хранения:
  - Групп
  - Преподавателей
  - Аудиторий
  - Сессий
  - Занятий
  - Дат расписания

### Модели данных
1. **Course (Занятие)**
   - Время начала
   - Продолжительность
   - Даты проведения
   - Название группы
   - Предмет
   - Преподаватель
   - Тип занятия
   - Подгруппа
   - Аудитория

2. **Schedule (Расписание)**
   - Название группы
   - Список занятий

### Конвертеры данных
Реализованы конвертеры для работы с:
- LocalTime (время)
- LocalDate (дата)
- Duration (продолжительность)
- List<LocalDate> (список дат)

### Пользовательский интерфейс
- Реализован экран расписания с использованием Material Design 3
- Отображение:
  - Времени занятия
  - Названия предмета
  - Типа занятия
  - Подгруппы
  - Преподавателя
  - Аудитории
- Поддержка навигации назад
- Подготовлена кнопка выбора даты (требует реализации)

### Функции работы с расписанием
- Получение расписания для группы
- Получение расписания для аудитории
- Получение расписания для преподавателя
- Обработка и сохранение расписания в базу данных

## Технические детали

### Зависимости
- Room Database для работы с SQLite
- Compose для UI
- Material Design 3 для современного интерфейса
- Gson для сериализации данных

### Архитектура
- Использование корутин для асинхронных операций
- Типобезопасные запросы к базе данных
- Чистая архитектура с разделением на слои:
  - UI
  - Data
  - Domain

## Как использовать

1. Инициализация базы данных:
```kotlin
val databaseBuilder = DatabaseBuilder(context)
```

2. Получение расписания:
```kotlin
viewModelScope.launch {
    val schedule = databaseBuilder.getScheduleForGroup("ИСП-101", LocalDate.now())
}
```

3. Отображение расписания:
```kotlin
ScheduleScreen(
    groupName = "ИСП-101",
    schedule = schedule,
    onDateChange = { newDate -> /* обработка смены даты */ },
    onNavigateBack = { /* навигация назад */ }
)
```
